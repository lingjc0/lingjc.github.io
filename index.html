my new blog

hexo常用命令
hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

hexo复合命令
hexo s -g #生成并本地预览
hexo d -g #生成并上传

输入hexo d无反应的请一定要检查_config.yml是否改写正确，且一定要是在执行hexo d命令的文件夹下的那个_config.yml文件

hexo检查秘钥是否连接成功的命令
$ ssh -T git@github.com # 注意邮箱地址不用改
此时回提示让你输入yes/no？
输入yes，若成功，则返回Hi xxx（xxx是GitHub用户名）
